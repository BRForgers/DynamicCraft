Overview

What is "DynamicCraft"?

"DynamicCraft" is a Chisel-Like Mod, that adds Building Blocks of all types. As other mods made by us, it requires BRForgersCore.

How I Install it?

  • If not yet, Install Minecraft Forge;
  • If you have, Delete any old versions of the ArchitectFriend.jar file from your ".minecraft/mods" folder.
  • If not yet, Download and put the BRForgersCore.jar file into your ".minecraft/mods" folder.
  • Download and put the ArchitectFriend.jar file into your ".minecraft/mods" folder.
  • Restart/Open Minecraft
